Introduccion a Programación Web Front-End

El programador front-end es el responsable de construir y maquetar todo lo que constituye la parte visible de una página web, aplicación o sistema. Para ello emplea distintas herramientas y lenguajes de programación. Se trata de profesionales que trabajan en la interfaz, pero que no se limita a concebir la parte gráfica de la misma, sino que también estudia las mejores maneras de hacer que todo sea entendible y funcional para los usuarios de la web o app.

Principales tecnologías de Front-End

  • HTML
  • CSS
  • JavaScript
  • React
  • Vue.js
